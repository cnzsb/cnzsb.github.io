title: '隐藏滚动条方法'
date: 2017-08-14 11:30:26
tags: [Tips, CSS]

---

隐藏滚动条方法：

```css
.container {
    /* only 'overflow-x: hidden' for firefox */
    overflow: hidden;

    .target {
        overflow-y: scroll;
        /* ie */
        -ms-overflow-style: none;
        
        /* chrome */
        &::-webkit-scrollbar {
          width: 0;
        }
        
        /* firefox (unreliable) */
        @-moz-document url-prefix() {
          & {
            margin-right: -16px;
            padding-right: 16px;
          }
        }
    }
}
```
